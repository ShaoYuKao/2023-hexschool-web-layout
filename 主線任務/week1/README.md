# 作業地雷

1. pixel perfect kata 寫死寬高
1. 兩個作業都需要正確使用 container
1. body 或最外層不可寫死 width: 1920px
1. 設計稿作業**不需使用 flex 語法**即可完成
1. 標題標籤須正確使用，例如：網頁有一個 h1 標籤、次標題依重要順序使用 h2, h3
1. 作業須符合此 [作業規範](https://hackmd.io/XbKPYiE9Ru6G0sAfB5PBJw)
