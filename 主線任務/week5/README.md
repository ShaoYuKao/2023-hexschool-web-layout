## 作業等級表

1. LV1：做一頁版型
1. LV2：做兩頁以上，並有設計 Modal 功能
1. LV3：全部頁面都做


## 作業地雷

- 注意 input readonly/disabled/value 用法
- 注意 label for 需對應 input id
- 頁數需以實際網站頁數計算
- 需至少完成 Lv1 作業需求
- 需使用線上圖示 Material Design Icons
- 作業須符合此[作業規範](https://hackmd.io/XbKPYiE9Ru6G0sAfB5PBJw)
